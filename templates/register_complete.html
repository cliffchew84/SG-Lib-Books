So now what?

Now that you have signed up for an account, you can start to search, save and identify where your favourite SG library books are in a way that is more intuitive!

  1. Start by searching and saving some books from Search NLB.
  2. You can go to Your Books to view all the books that you have saved.
  3. Go to the SG Libraries to find the availability of your saved books at a library level.
  4. You can update all books' availability by clicking on Update Avails, or update the books individually.
  5. Book titles also link to the book's NLB page for more information about them.
  6. I will add more features to this app, so do keep a lookout! Feel free to reach out to me if there are things you want to see here!
